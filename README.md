tag-linker
==========

Tool to be used to link "elements" with tags or links


Dependencies
============

* Qt 5.X
* CMake
* C++11 +
* boost (only for boost::split (algorithm/string)) should be removed soon
* qxtglobalshortcut: https://github.com/hluk/qxtglobalshortcut (build and install)
